# Deepfake Detection Tool

A deep learning project to detect deepfake images using a CNN model.

## Dataset
- Source: [Kaggle - Deepfake and Real Images](https://www.kaggle.com/datasets/manjilkarki/deepfake-and-real-images)

## What this project does
- Loads and preprocesses real and fake images
- Builds a Convolutional Neural Network (CNN)
- Trains the model to classify images as Real or Fake
- Plots training accuracy
- Saves the trained model
